Media Feature – Hover

none

The primary input mechanism cannot hover at all or cannot conveniently hover (e.g., many touch devices emulate hovering only via long-press), or there is no pointing input mechanism. 

@media (hover: none) {
  body::before {
    display: block;
    content: 'The device does NOT support hover';
  }
}

hover

The primary input mechanism can conveniently hover over elements. 

@media (hover: hover) {
  body::before {
    display: block;
    content: 'The device does support hover';
  }
}
