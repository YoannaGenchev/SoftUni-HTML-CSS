Radio

The input type="radio" element allows a single value to be selected out of multiple choices with the same name value.

Attribute – type="radio" name="fruits" id="apple"

A radio button, grouped by the name attribute.

<p>
  <input type="radio" name="fruits" id="apple">
  <label for="apple">Apple</label>
</p>
